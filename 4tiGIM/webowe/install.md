## Użyteczne komendy

### Tworzenie projektu
```console
npx create-react-app nowa-app --template typescript

```

### Odtwarzanie katalogu node-modules (po pobraniu z gitHuba np)

```console
npm i
//lub
npm install

```

### dodawanie pakietów

```console
npm i bootstrap

npm i react-router
npm i react-router-dom

```
